## 下载

### Docker

```
docker pull apachecn0/apachecn-vue-zh
docker run -tid -p <port>:80 apachecn0/apachecn-vue-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-vue-zh
apachecn-vue-zh <port>
# 访问 http://localhost:{port} 查看文档
```
